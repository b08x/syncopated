# Audio Production

Audio system and DAW configuration roles

## Roles

The following roles are included in this collection:

- alsa
- pulseaudio
- pipewire
- jackd
- sonic-pi
- daw
- deadbeef

## Usage

Install this collection using:

```bash
ansible-galaxy collection install custom.audio_production
```

## Requirements

- Ansible 2.9 or later
